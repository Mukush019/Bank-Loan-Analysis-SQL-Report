# Loan Analysis Report 
---
## Project Overview
This project is focused on credit risk analysis using MySQL to analyze financial loan data. It provides insights into loan applications, disbursments, and risk metrics. The aim of this project is to help financial institutions manage risk by understanding loan portfolio performance.
